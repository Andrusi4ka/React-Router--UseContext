# Вступ до маршрутизації в React з використанням React Router

### 1. Ініціалізація проекту:

- Створіть новий React проект за допомогою Vite:
```cmd
npm create vite@latest my-react-router-app -- --template react
```

- Перейдіть до каталогу проекту:
```cmd
cd my-react-router-app
```

- Встановіть залежності:
```cmd
npm install
```

- Встановіть бібліотеку react-router:
```cmd
npm install react-router
```

### 2. Створення компонентів:

- У директорії `src/components` створіть три компоненти: `Home.jsx`, `About.jsx`, `Contact.jsx`.
- Кожен компонент повинен відображати унікальний текст, який ідентифікує його призначення (наприклад, "Це домашня сторінка", "Це сторінка про нас", "Це контактна сторінка").

### 3. Налаштування маршрутизації:

- У файлі `src/App.jsx` імпортуйте та використовуйте `BrowserRouter` для ініціації маршрутизації.
- Налаштуйте маршрути за допомогою компонентів `Route`:
- / → Home
- /about → About
- /contact → Contact
- Використовуйте компонент Link для створення навігаційних посилань у головному меню.

### 4. Використання NavLink:

- Замініть `Link` на `NavLink` у головному меню.
- Налаштуйте NavLink, щоб активне посилання виділялося стилем або класом.

### 5. Тестування проекту:

- Запустіть проект:

```cmd
npm run dev
```

- Переконайтеся, що перехід між сторінками відбувається без перезавантаження браузера.

### 6. Деплоймент:

- Розгорніть додаток на платформі хостингу (наприклад, Vercel або Netlify).
